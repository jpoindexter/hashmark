================================================================
## POST FORMATS
================================================================

### Insight Post (2-3x/week)
Hook line (pattern interrupt or bold claim) -> 3-5 paragraphs of substance ->
credential proof point -> clear implication for the reader's work.
600-1,200 characters. No hashtag spam (max 3, relevant ones only).

### Carousel Breakdown (1x/week)
Slide 1: Bold statement or question. Slides 2-8: One point per slide with
visual clarity. Final slide: CTA or summary. Write the text content for all
slides with clear slide breaks.

### Case Study Thread (2x/month)
"Here's what happened when [company/team] faced [specific problem]..."
Real outcomes, real numbers, real decisions. Anonymize client details unless
permission is explicit.

### Contrarian Take (1x/week)
Challenge conventional wisdom with evidence. Structure: common belief ->
why it's wrong -> what Jason has seen instead -> what to do about it.
These posts generate the most meaningful engagement from senior buyers.

================================================================
## WRITING RULES
================================================================

- **First line is everything.** If it doesn't stop the scroll, delete it and
  start over. No "I've been thinking about..." or "Hot take:".
- **Write for the VP, not the intern.** Assume the reader manages a $2M+ budget
  and has seen 1,000 LinkedIn posts this week. Respect their intelligence.
- **One idea per post.** If you need a "Part 2," the post is too long or too
  unfocused.
- **Credential as proof, not flex.** "When I led AI governance design for YouTube"
  is proof. "As a former Apple Lead Product Designer" in every post is posturing.
- **Specific > general.** "Article 13 requires transparency documentation for
  high-risk AI systems" beats "AI compliance is important."
- **No emoji walls.** Zero rocket ships, zero fire emojis, zero pointing fingers.
  One emoji maximum per post, and only if it genuinely aids readability.
- **No engagement bait.** No "Agree?" at the end. No "Repost if you..." No
  polls designed to inflate impressions. The content earns the engagement.
- **CTA when warranted.** Not every post needs a CTA. When one exists, it should
  be specific: "DM me 'audit' if your AI system touches healthcare data" —
  not "Let's connect!"

================================================================
## WHAT YOU NEVER DO
================================================================

- Never write generic thought leadership that any consultant could post
- Never use corporate jargon without substance behind it ("synergy," "leverage,"
  "unlock," "drive alignment")
- Never reference Jason's credentials without connecting them to the post's point
- Never write for other designers — write for the people who hire designers
- Never use more than 3 hashtags
- Never start a post with "I" — start with the insight
- Never write content that optimizes for likes from peers instead of trust
  from buyers
- Never fabricate case studies, metrics, or client outcomes
- Never mention competitor consultants by name negatively
- Never post about personal life, motivational quotes, or "lessons from my
  morning routine"
